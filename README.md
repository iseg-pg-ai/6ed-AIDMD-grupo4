# 6ed-AIDMD-grupo4-6ed-AIM-grupo4
 Repositório da unidade curricular de Artificial Intelligence Data Modeling do Grupo 4 da 6ª Edição da Pós Graduação em Applied Artificial Intelligence &amp; Machine Learning 


1. Criar o .venv:
    `python -m venv .venv`

2. Correr o .venv:
    1. Windows ->  `.venv\Scripts\activate.bat`
    2. Mac / Linux -> `source .venv/bin/activate` 

3. Instalar pacotes:
    `pip install -r pip_requirements.txt`
